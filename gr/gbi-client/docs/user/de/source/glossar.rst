Glossar
=======

Rasterdaten
    Enthalten zum Beipsiel Pläne oder Luftbilder. Sie enthalten keine weiteren Informationen sondern bestehen aus den dargestellten Pixeln.  

Vektordaten
    Beinhalten informationen zu Punkten, Flächen und Linien. Diese können zum Beispiel in einem Geoinformationsystem geladen und über Rasterdaten gelegt werden.
